# Voice Samples for DungeonMaister

Place a WAV audio file here to use as a voice reference for Chatterbox TTS voice cloning.

## How to use

1. Record or find a 3-10 second WAV clip of the voice you want the DungeonMaister to sound like
2. Save it as `dm_voice.wav` in this directory
3. Restart the backend — Chatterbox will automatically clone that voice

## Notes

- The clip should be clean speech with minimal background noise
- 3-10 seconds is the sweet spot — longer isn't necessarily better
- If no `dm_voice.wav` is present, Chatterbox will use its default voice
- The same voice will be used across all languages (Chatterbox handles cross-language transfer)
